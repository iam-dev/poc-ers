# POC ERS
Proof Of Concept Ethereum Reality Service, based on https://ethereum-magicians.org/t/a-proposal-for-the-ethereum-reality-service/9694

## Overview

### Installation

```console
forge install && yarn install
yarn prepare
```

Make sure all @solidstate libary are install
```console
yarn pre-build

```

### Test

```console
yarn test
```
